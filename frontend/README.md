# 🚦 bongo-traffic-ai

Tanzanian traffic detection powered by YOLO `best.pt`.

**Detects:** Bajaj · Bodaboda · Daladala · Car · Truck · Motorcycle

## Setup
Set `BACKEND_URL` in HuggingFace Spaces → Settings → Repository Secrets:
```
BACKEND_URL = https://bongo-traffic-ai.up.railway.app
```
